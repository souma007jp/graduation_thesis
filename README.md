# graduation_thesis
以下に各ファイルの内容を示す。

## str_t_vs_pc.py
このコードは「各ゲノム状態の粒子分布の時間変化」をプロットするコードである。
N=2、つまり2粒子共感染モデル専用であり、Nを変更すると正常に動作しない。
target_muではプロットしたい変異率を設定できる。
num_genomesは分節ゲノムの本数を意味し、これを変更すると分節ゲノム数に応じてプロットが生成される。
t1は最終時刻を表し、0からt1までの時間変化をプロットする。
kでは増殖項の係数kを任意に決定できる。

このコードを実行するときは、requirements.txtを満たすPython環境で

cd "str_t_vs_pc.pyのあるファイルのパス"
python str_t_vs_pc.py

を実行すればよく、プロットは256_t_vs_pc_N2_mu{target_mu}.pngとしてdpi=300で保存される。

## str_N2_mu_vs_pc.py
このコードはt=t1における厳密モデルの「変異率に対する各ゲノム状態の粒子分布変化」および「変異率に対する減少項の係数d(t)の変化」をプロットするコードである。
N=2、つまり2粒子共感染モデル専用であり、Nを変更すると正常に動作しない。
num_genomesは分節ゲノムの本数を意味し、これを変更すると分節ゲノム数に応じてプロットが生成される。
t1は最終時刻を表し、この時間における粒子数やd(t)をプロットするため、定常状態へと達するに十分な時間を設定する必要がある。
kでは増殖項の係数kを任意に決定できる。
